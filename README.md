# id Mobile RPG Editor
Распаковщик и запаковщик ресурсов мобильных J2ME RPG игр от id Mobile.

## Поддержка
### Распаковка
| Игра | Текстуры | Палитры | 
| :---: | :---: | :---: |
| Doom RPG | Полная | Полная |
| Orcs & Elves | Полная | Частичная |
| Orcs & Elves II | Полная | Частичная |

### Запаковка
| Игра | Текстуры | Палитры | 
| :---: | :---: | :---: |
| Doom RPG | Полная | Полная |
| Orcs & Elves | Не поддерживается | Не поддерживается |
| Orcs & Elves II | Не поддерживается | Не поддерживается |

## Установка
Распокавать в любое удобное место. Это портативное приложение.

## Как пользоваться
1. Выберите **jar** файл поддерживаемое игры.
2. Нажмите "Распаковать". Программа создаст в директории игры папку с названием обнаруженной игры и подпапку **Unpacked**, и извлечет в неё **bin** файлы из **jar**. Далее выполниться распаковка из **bin** файлов ресурсов игры в соотвествующие папки в формате **png**.

***Пример для "Doom RPG.jar"***
```
Doom RPG/
└── Unpacked/
    ├── Textures/
    └── Palettes/
    └── wtexels.bin
    └── palettes.bin
```

3. Отредактируйте **png** файлы в любом удобным для вас графическом редакторе.
4. Нажмите "Запаковать". Программа выполнить запаковку текстур и палитр в соотвествующие **bin** файлы. Далее выполниться замена **bin** файлов в ранее выбраном **jar** файле игры.
